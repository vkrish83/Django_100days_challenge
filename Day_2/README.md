## 🖥️ How to Run:
1. Activate virtual environment:
   ```bash
   source myenv/bin/activate  # Mac/Linux
   myenv\Scripts\activate     # Windows
   ```
2. Run the server:
   ```bash
   python manage.py runserver

   ```
3. Visit ```http://127.0.0.1:8000/hello/``` in your browser.
